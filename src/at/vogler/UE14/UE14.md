### Aufgaben

*   Lösche Zeichen an Position: Schreibe eine Methode  
    `String deleteCharAtPos(String s, int pos) { … } ,`  
    die aus dem String s EIN Zeichen an der Position pos „löscht“ und den verkürzten String zurückliefert. Fehlerhafte Angaben von pos sollen ignoriert (das Programm darf auch nicht abstürzen) und der gesamte String zurückgeliefert werden.  
    `deleteCharAtPos(“Hallox 1XI“, 5) → „Hallo 1XI“`  
    `deleteCharAtPos(“Hallo Welt“, 18) → „Hallo Welt“`
*   zähle Zeichen aus Vorrat: Schreibe eine Methode  
    `int zaehleZeichenAusVorrat(String s, String vorrat) { … } ,`  
    die zählt, wie viele Zeichen von s aus dem Vorratsstring vorrat sind.  
    `zaehleZeichenAusVorrat (“Hallo 1XI“ , "aeiou") → 2`  
    `zaehleZeichenAusVorrat (“*code*123#“ , "?%*!#$“) → 3`
*   besteht nur aus Vorrat: Schreibe die Methode  
    `boolean nurAusVorrat(String s, String vorrat) { .. },`  
    die prüft, ob der String s nur aus Zeichen besteht, die im String vorrat definiert sind.  
    `nurAusVorrat (“1232432“, „1234“) → true`  
    `nurAusVorrat (“12352432“, „1234“) → false`  
    4\. erzeuge Zufallsstring aus Vorrat: Schreibe die Methode  
    `String createRandomString(String vorrat, int len) { .. },`  
    die einen String mit der Länge len erzeugt, der nur aus zufällig gewählten Zeichen aus dem String vorrat besteht.  
    `createRandomString (“1234“, 5) → (z.B.: 24211)`
*   5\. Groß/Klein abwechselnd: Schreibe die Methode  
    `String createUpperLower(String s) { .. },`  
    die einen String erzeugt, bei dem die Zeichen abwechselnd groß bzw. klein sein sollen. Es soll immer mit einem Großbuchstaben begonnen werden. Der Einfachheit halber kümmern wir uns nicht um Zeichen, für die es keine Groß-/Kleinschreibung gibt (Ziffern, Sonderzeichen, …)  
    `createUpperLower (“hallo welt“) → “HaLlO WeLt”`  
    `createUpperLower (“a1a2a3xxx“) → “A1A2A3XxX”`  
    Tipp: In der Klasse Character gibt es einige sehr nützliche Funktionen, die bei der Lösung dieses Beispiels sicher hilfreich sind ….  
    Tippe in IntelliJ in einer leeren Zeile folgendes ein: Character.to und betrachte die angebotenen Auswahlmöglichkeiten!
*   Schreibe die Methode  
    `String cut(String s, int length) { .. }`  
    die den Text auf die Länge length kürzt.
    *   Falls length <= 3 ist, soll s unverändert zurückgegeben werden
    *   Falls s kürzer als oder gleich lang wie length ist, so soll s unverändert zurückgegeben werden.
    *   Falls s länger als length ist, so soll s so abgeschnitten werden, dass hinter den length-3 Zeichen drei Punkte „...“ angehängt und zurückgegeben werden. (D.h. insgesamt werden length-Zeichen zurückgeliefert, wobei die letzten drei Zeichen durch Punkte ersetzt werden.)
*   Beispiele:  
    `cut(“Ein langer Text“, 10) → “Ein lan...“`  
    `cut(“Ein anderer Text“, 20) → “Ein anderer Text“`
