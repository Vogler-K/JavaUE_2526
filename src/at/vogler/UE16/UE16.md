### Aufgaben

*   **printArray:** Schreibe die Methoden  
    `void printArray(int[] a) { .. }`  
    `void printArray(double[] a) { .. }`  
    für das Ausgeben eines Arrays. Das Array soll in Java-Notation ausgegeben werden:  
    vor dem eigentlichen Array soll das Zeichen ‘ {‘ , danach alle Elemente des Arrays durch Beistrich getrennt, am Schluss noch ‘ }‘ stehen  
    Ausgabe-Beispiele: `{1, 12, 5, 89, 18, 37, 19}` oder `{2.34, 8.71, -123.5}`;  
    Verwende diese Methode(n), um die in den weiteren Beispielen erzeugten Arrays auszugeben!  
    Testen kannst Du diese Methode z.B.: mit diesen Anweisungen:  
    `int[] check = {1,12,5,89,18,37,19};`  
    `printArray(check);`
*   **Zahlenreihen im Array:**  
    Definiere 3 integer-Arrays mit der jeweils angegebenen Menge von Elementen und fülle sie mit Hilfe einer Schleife mit den Zahlen der folgenden Zahlenreihen:
    *   Die Zahlen 1, 2, … 25. -> Anzahl Elemente = 25
    *   Die Zahlen 5, 10, 15, 20, … 100. -> Anzahl Elemente = 20
    *   Die Zahlen -5, -4, -3, … 4, 5. -> Anzahl Elemente = 11  
        Gib die erzeugten Arrays in der Konsole aus!!
*   **Zufallszahlen1:** Schreibe die Methode  
    `int[] makeRandomArray() { .. },`  
    die ein 30-Elemente großes int\[\]-Array mit Zufallszahlen zwischen 1 und 10 erzeugt und zurückliefert.  
    Erzeuge mehrere Zufallsarrays ( `int [] zuf1 = makeRandomArray();` .. ) und gib diese Arrays mit Hilfe von `printArray(..)` aus.
*   **Zufallszahlen2:** Schreibe die Methode  
    `int[] randomArray(int anzahl, int von, int bis) { .. }`  
    Die Methode soll ein Zufallszahlen-Array mit folgenden Eigenschaften zurückgeben:  
    Das erzeugte Array soll **anzahl** Elemente haben. Die Elemente sollen eine zufällige Zahl zwischen **von** und **bis** sein. (von und bis jeweils inklusive.)  
    Teste die Methode mit  
    `int[] zufall = randomArray(30, -5, 5);`  
    `printArray(zufall);`  
    `printArray(randomArray(10, -1, 1));`  
    `printArray(randomArray(10, 1, 3));`  
    `printArray(randomArray(10, -3, -1));`
