Folgende Methoden aus der Theorie kommen zur Anwendung:

*   .length() , .charAt(..)
*   .equals(..) , .equalsIgnoreCase(..)
*   .indexOf(..) , .toLowerCase() , .toUpperCase()

### Aufgaben

*   jedes zweite Zeichen: Schreibe eine Methode  
    `String jedesZweite(String s1) { … } ,`  
    die einen Ergebnisstring zurückliefert, der nur das 1., 3., 5., …(also jedes zweite) Zeichen des übergebenen Strings s1 enthält.  
    (Aus „Hallo Welt“ wird „HloWl“)
*   umdrehen: Schreibe eine Methode  
    `String reverseString(String s1) { … } ,`  
    die den übergebenen String „umdreht“, also die Zeichen von hinten nach vorne  
    anordnet.  
    (Aus „Hallo Welt“ wird „tleW ollaH“)
*   Leerzeichen löschen: Schreibe eine Methode  
    `String deleteBlanks(String s1) { … } ,`  
    die aus dem übergebenen String s1 alle Leerzeichen löscht und das Ergebnis zurückliefert.  
    (Aus „Hallo Welt wie gehts“ wird „HalloWeltwiegehts“)
    *   Anleitung: In einer Schleife muss jedes Zeichen analysiert werden, Leerzeichen werden NICHT in den Ergebnisstring aufgenommen
*   Palindrom: Ein Palindrom ist eine Zeichenkette, die von vorn und von hinten gelesen dasselbe ergibt. Palindrome können einzelne Worte (Otto, Lagerregal, Anna ) oder auch Wortgruppen bzw. Sätze sein („Trug Tim eine so helle Hose nie mit Gurt“).  
    Schreibe eine Methode  
    `boolean istPalindrom(String s1) { … } ,`  
    die überprüft, ob der übergebene String s1 ein Palindrom ist oder nicht.  
    Hinweis: Leerzeichen sowie Groß-/Kleinschreibung müssen ignoriert werden! Kopiere s1 in eine zweite Stringvariable, bearbeite die beiden Strings entsprechend und vergleiche sie. Verwende die bereits erstellten Methoden reverseString() und deleteBlanks()
*   ist enthalten: Schreibe eine Methode  
    `boolean istEnthalten(String s1, String s2) { … } ,`  
    die überprüft, ob der String s2 im String s1 enthalten ist.
*   wie oft enthalten: Schreibe eine Methode  
    `int zaehleVorkommen(String s1, String s2) { … } ,`  
    die überprüft, wie oft der String s2 im String s1 vorkommt.  
    Hinweis: verwende in einer Schleife .indexOf(String, index) so, dass nach jedem Treffer der index für die weitere Suche entsprechend angepasst wird.  
    Bsp: zaehleVorkommen(“Raubbau“,“au“) → liefert 2
