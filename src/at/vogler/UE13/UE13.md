Folgende Methoden aus der Theorie kommen zur Anwendung:

*   .length() , .charAt(..)
*   .equals(..) , .equalsIgnoreCase(..)
*   .indexOf(..) , .substring()

### Aufgaben

*   1\. String und Substring:  
    lies wiederholt mittels PopUp-Fenster so lange Zeichenketten ein, bis das Wort “quit“ eingegeben wird. Für jeden Eingabestring sollen die unten definierten Ausgaben durchgeführt werden (“quit“ beendet die Ausführung und führt zu keinen weiteren Ausgaben). Verwende jeweils die substring()-Methode.
    *   Ausgabe 1:  
        Wenn die Eingabe mindestens drei Zeichen hat, soll das zweite und dritte Zeichen ausgegeben werden.
    *   Ausgabe 2:  
        Wenn die Eingabe mindestens zwei Zeichen hat, sollen die letzten zwei Zeichen ausgegeben werden.
    *   Ausgabe 3:  
        Wenn die Eingabe mindestens drei Zeichen hat, sollen die ersten zwei und die letzten drei Zeichen ausgegeben werden.
*   lösche Teilstring: Schreibe die Methode  
    `String delStringAtPos(String s, int startPos, int len) { .. },`  
    die einen Ergebnisstring zurückliefert, bei dem aus dem String s ein Teilbereich herausgelöscht wurde, der durch die Anfangsposition startPos und durch seine Länge len festgelegt ist. Das Programm soll nicht abstürzen, wenn startPos und/oder len zu ungültigen Situationen führen, sondern den Originalstring zurückliefern (Anleitung: Hier sind passende Substrings zu ermitteln und zum Ergebnis zusammenzufügen …..)  
    delStringAtPos("Hallo Leute!", 6, 5) liefert: Hallo !
*   enthält Ziffern: Schreibe eine Methode  
    `boolean enthaeltZiffern(String s) { … } ,`  
    die genau dann true zurückliefert, wenn der übergebene String mindestens eine Ziffer ( ‘0‘ .. ‘9‘) enthält.  
    `enthaeltZiffern(“Hallo 1AI“) → true`  
    `enthaeltZiffern(“Hallo Welt“) → false`
*   enthält nur Ziffern: Schreibe eine Methode  
    `boolean enthaeltNurZiffern(String s) { … } ,`  
    die genau dann true zurückliefert, wenn der übergebene String s zur Gänze aus Ziffern ( ‘0‘ .. ‘9‘) besteht.  
    `enthaeltNurZiffern(“Hallo 1AI“) → false`  
    `enthaeltNurZiffern(“6270983“) → true`  
    `enthaeltNurZiffern(“627x983“) → false`
*   zähle Kleinbuchstaben: Schreibe die Methode  
    `int countLowerCaseLetters(String s) { .. },`  
    die zählt, wie viele Zeichen im String s Kleinbuchstaben sind. Die deutschen Umlaute sollen nicht mitgezählt werden.  
    `countLowerCaseLetters(“Hallo 1aI“) → 5`
*   BB-Sprache: Schreibe die Methode  
    `String bbSprache(String satz) { ... }`  
    Diese Methode soll den satz in der BB-Sprache liefern. In dieser „Sprache“ wird jeder Selbstlaut und Umlaut verdoppelt und dazwischen ein 'b' eingefügt.  
    Aus "Hallo, Erde!" wird "Haballobo, Eberdebe!"  
    Achte auf korrekte Groß- und Kleinschreibung!
