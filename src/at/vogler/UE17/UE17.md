### Aufgaben

Teste die folgenden Methoden wenn möglich jeweils mit einem Zufallsarray, das mit der entsprechenden Methode aus der vorigen Übung angelegt wurde.

*   **indexOf (1):** Schreibe die Methode  
    `int indexOf(int [] ia, int n) { .. }`  
    die die Zahl **n** im Array **ia** sucht (das erste Vorkommen) und die Position (beginnend ab 0) zurückgibt. Falls **n** nicht gefunden wurde, soll -1 zurückgegeben werden.  
    Teste mit einem Zufallsarray, das mit der entsprechenden Methode aus der vorigen Übung angelegt wurde.
*   **indexOf (2):** Schreibe die Methode  
    `int indexOf(int [] ia, int n, int pos) { .. }`  
    die die Zahl **n** im Array **ia** ab der Position pos sucht (das erste Vorkommen) und die Position zurückgibt. Falls **n** nicht gefunden wurde, soll -1 zurückgegeben werden.  
    **Anmerkung:** Die beiden Methoden aus Beispiel 1 und 2 haben den gleichen Namen, allerdings eine unterschiedliche Parameterliste. Das ist erlaubt (und in Java auch häufig zu finden). Welche Methode vom Programm ausgeführt wird, hängt von den Parametern ab, die beim Aufruf der Methode verwendet werden!!!
*   **addNumber:** Schreibe die Methode  
    `int[] addNumber(int[] ia, int n) { .. },`  
    die jedes Element von **ia** um die Zahl **n** erhöht.  
    Teste z.B. mit:  
    `int [] a = randomArray(10, 0, 9);`  
    `printArray(a);`  
    `a = addNumber(a, 100);`  
    `printArray(a);`
*   **Array kopieren:** Schreibe die Methode  
    `double[] copyArray(double[] a) { .. }.`  
    Sie soll eine Kopie des Arrays a zurückgeben (d.h. ein Array mit derselben Länge und denselben Werten).
*   **String analysieren:** Schreibe die Methode  
    `boolean[] stringToBooleanArray(String s) { .. },`  
    die ein Boolean-Array mit der Länge s.length() zurückgibt, in dem für jedes Leerzeichen das dazugehörige Element auf false gesetzt wird.  
    Beispiel:  
    `stringToBooleanArray("I am") → {true, false, true, true}`
*   **Array für die Rückgabe mehrerer Werte:** Schreibe die Methode  
    `int[] summen(boolean[] ba) { .. },`  
    die die Summe aller **true** und die Summe aller **false** Elemente von Array **ba** ausrechnet und als **int**\-Array mit zwei Elementen zurückgibt.  
    z.B.:  
    `boolean [] bool = new boolean[] {true, true, false, true, false};`  
    `int [] sums = summen(bool); // sums → {3,2}`
*   **BONUSAUFGABE(anspruchsvoll!) Doppelte löschen:** Schreibe die Methode  
    `int[] deleteEquals(int[] ia) { .. } ,`  
    die aus dem Array **ia** alle doppelten Zahlen herauslöscht und das (wahrscheinlich kürzere) Array zurückgibt. Die Reihenfolge muss beibehalten werden!  
    Dafür sind sicher mehrere Zwischenschritte notwendig.  
    Beispiel:  
    Das Array {1, 3, 3, 1, 2, 1, 5} nach dem Löschen der doppelten Zahlen: {1, 3, 2, 5}.
