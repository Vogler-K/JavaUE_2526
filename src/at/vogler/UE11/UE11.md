Folgende Methoden aus der Theorie kommen zur Anwendung:

*   .length()
*   .charAt(..)
*   .equals(..) 

zum Vergleichen von Strings :

*   if (eingabe.equals(“quit“) { … } // nicht eingabe == “quit“ !!

### Aufgaben

*   String und char: Schreibe die Methode void stringAnalyzer() mit folgender Aufgabe:  
    Lese mittels PopUp-Fenster so lange Texte ein, bis das Wort “quit“ eingegeben wird (d.h.: die Benutzereingabe ist in einer Schleife zu implementieren). Gib von jedem eingegebenen Text (jede Ausgabe in einer neuen Zeile) die folgenden Zeichen aus:
*   *   Letztes Zeichen: Wenn die Eingabe mindestens ein Zeichen hat, soll das letzte Zeichen ausgegeben werden.
*   Zweites Zeichen: Wenn die Eingabe mindestens zwei Zeichen hat, soll das zweite Zeichen in der Konsole ausgegeben werden.
*   Ersten beiden Zeichen: Wenn die Eingabe mindestens zwei Zeichen hat, soll das erste und zweite Zeichen ausgegeben werden. (in einer Zeile)
*   Vorletztes Zeichen: Wenn die Eingabe mindestens zwei Zeichen hat, soll das vorletzte Zeichen ausgegeben werden.
*   Zweites und vorletztes: Wenn die Eingabe mindestens zwei Zeichen hat, soll das zweite und das vorletzte Zeichen ausgegeben werden. (in einer Zeile)
*   untereinander: Schreibe die Methode  
    `void verticalString(String s) { … }`  
    die dafür sorgt, dass der übergebene String zeichenweise untereinander ausgegeben wird.  
    Aus „hallo“ wird also:  
    h  
    a  
    l  
    l  
    o  
    Hinweis: In einer for-Schleife muss mit charAt(..) jedes einzelne Zeichen des Strings ermittelt und ausgegeben werden.
*   mitStern: Schreibe die Methode  
    `void withStars(String s) { … }`  
    die dafür sorgt, dass der übergebene String so ausgegeben wird, dass nach jedem Zeichen des Strings ein Stern angezeigt wird.  
    Aus „hallo“ wird also h\*a\*l\*l\*o … der letzte Stern soll weggelassen werden!  
    Wie beim vorigen Beispiel ist auch hier eine Schleife notwendig …..
*   String ausgeben: Schreibe eine Methode  
    `void displayString(String s1) { … } ,`  
    die den übergebenen String wie unten abgebildet ausgibt:  
    (Beispielstring: „Die ersten 5 Tage nach dem Wochenende sind die Schlimmsten 😊“  
    Ausgabe:  
    `Die ersten 5 Tage nach dem Wochenende sind die Schlimmsten 😊`   
    `0         1         2         3         4         5         6`  
    `0123456789012345678901234567890123456789012345678901234567890`  
    Es sollen also unter der Zeichenkette die Indexpositionen der einzelnen Zeichen (bis zur letzten Position – also bis zum Ende des Strings) ausgegeben werden. Du brauchst für diese Zahlenausgabe zwei Schleifen!  
    Lies für diese Methode mehrere kurze und längere Strings ein und teste Dein Ergebnis!!
