### Aufgaben

*   Schreibe die Methode  
    `String removeNonLetters (String s) { .. }`  
    die alle Zeichen aus dem String s herauslöscht, die keine Buchstaben sind.  
    Tipp: Mittels der Methode Character.isLetter(char ch) kann überprüft werden, ob das Zeichen ch ein Letter ist.  
    Beispiele:  
    `Character.isLetter('ü') → true,`  
    `Character.isLetter('+') → false`
*   nur ein Leerzeichen: Schreibe die Methode  
    `String toSingleSpace(String text) { ... }`  
    Diese Methode soll den text so verkürzen, dass mehrfache Leerzeichen durch ein einzelnes ersetzt werden  
    Beispiel:  
    Für "Ein Text mit viel Zwischenraum" soll diese Methode das Ergebnis "Ein Text mit viel Zwischenraum" liefern.
*   Schreibe die Methode  
    `String firstUpperCase(String word) { .. }`  
    die den Text word mit dem ersten Zeichen in Groß- und allen anderen Zeichen in Kleinbuchstaben zurückgibt.
    *   Falls word ein Leerstring ist, so soll ein Leerstring zurück gegeben werden.
    *   Falls word mehr als ein Wort enthält, so soll nur das erste Wort mit einem Großbuchstaben versehen werden, alle anderen sind klein zu schreiben
*   Schreibe die Methode  
    `String createRandomPassword (int length) { .. } ,`  
    die ein „starkes“ Passwort mit Zufallszeichen erzeugt.
    *   Das Passwort soll length Zeichen haben.
    *   Es soll aus den Zeichen „a-zA-Z0-9“ bestehen.
    *   Jedoch ohne dem kleinen 'L' bzw. dem großen 'O', da diese Zeichen schlecht von 1 bzw 0 unterschieden werden können.
    *   Tipp: Ein Passwort mit Zufallszeichen ist ziemlich unknackbar.
*   Schreibe die Methode  
    `public static String rot13(String s) { .. } ,`  
    die den Text s mit dem ROT13-Algorithmus verschlüsselt.
    *   Dabei werden alle 26 Buchstaben (A-Z bzw. a-z) um 13 Stellen verschoben.
        *   A-M (bzw. a-m) werden um 13 Stellen nach hinten verschoben.
        *   N-Z (bzw. n-z) werden um 13 Stellen nach vorne verschoben.
    *   Alle anderen Zeichen bleiben unverändert.
    *   Die Groß-/Kleinschreibung bleibt unverändert.
        *   Tipp: rot13(rot13(s)) → s (d.h. verschlüsselt man s zweimal bekommt man wieder s.)
